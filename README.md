# ContactHub 📇

A modern and responsive contact management web application built with **HTML, CSS, and JavaScript**. ContactHub helps users efficiently organize, manage, and access their contacts with a simple and intuitive interface.

## Features

### 📌 Contact Management

* Add new contacts
* Read and display saved contacts
* Update existing contacts
* Delete contacts

### 🔍 Smart Search

Search contacts instantly by:

* Name
* Email
* Phone Number

### ⭐ Favorites & 🚨 Emergency Contacts

* Mark contacts as **Favorite**
* Mark contacts as **Emergency Contacts**
* Easily add or remove contacts from these categories at any time

### ✅ Form Validation

Robust client-side validation ensures users enter valid:

* Names
* Email addresses
* Phone numbers

### 💾 Local Storage Support

All contacts are stored in the browser using **Local Storage**, allowing data to persist between sessions without requiring users to re-enter their contacts.

### 📞 Quick Actions

Each contact includes action buttons for:

* Calling the contact directly
* Sending an email instantly

### 📱 Responsive Design

Fully responsive interface that provides a seamless experience across:

* Desktop
* Tablet
* Mobile devices

### 🎨 User-Friendly Experience

Designed with a focus on:

* Uses SweetAlert2 for interactive alerts and confirmations
* Confirmation dialog before deleting contacts
* Clean UI
* Easy navigation
* Fast contact management
* Improved usability and accessibility

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* SweetAlert2

## Project Goal

ContactHub was created to provide a lightweight and efficient contact management solution that runs entirely in the browser, offering quick access to important contacts while maintaining a smooth and user-friendly experience.
